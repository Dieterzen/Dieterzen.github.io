Advanced Useless Box

Dieter Zenner.

Waarom?

Dit is mijn 2de keer dat ik in mijn zesde jaar zit. Vorig jaar besloot ik om mijn eigen game te maken, dit is niet goed afgelopen omdat ik er telaat aan ben begonnen. Dit jaar wil ik iets fysiek maken. Dus in het begin van het jaar toen ik 3 voorstellen moest doen koos ik allemaal fysieke projecten.

Mijn 3 voorstellen waren:

Zelfoplaadbare lichtgevende schoenen;

S'werelds grootste Gameboy;

En de advanced useless box.

Hoe ga ik dit doen?

Ik moet de meeste info zelf opzoeken en kan bij sommige dingen een handleiding volgen.

Deadline.

Op de opendeurdag moeten we onze GIPs kunnen voorstellen.
